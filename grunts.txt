
* Grunt
Its a tool to automate tasks in projects, like:
- Concatenate css/js files.
- Minify css/js files.
- Handle images.
- Compile sass files.
- Unit test.
- Building final project.
- Set watcher for changes.
- Provide a server.


* Having Grunt Cli installed
/> npm install -g grunt-cli


* Installing a dependency
/> npm install LIB_NAME --save


* Installing a dependency used only in development
/> npm install LIB_NAME --save-dev


* Install grunt workflow into a project
/> npm init
/> npm install grunt --save-dev
/> touch Gruntfile.js
